# Nipret

[English](README.md) · **Русский**

Простая веб-страница для роутера на OpenWrt, которая **разблокирует сайты и приложения двумя способами** —
через **VPN** и через **обход DPI** — кнопками в один клик для Telegram, YouTube, Netflix, ChatGPT и
других. Nipret — удобная надстройка над **[nikki](https://github.com/nikkinikki-org/OpenWrt-nikki)** (VPN
Mihomo) и **[zapret2](https://github.com/bol-van/zapret)** (обход DPI). Управляет тем из двух, что у тебя
установлено.

> *Nipret = **Ni**kki + за**pret**.* (Пакет/репозиторий по-прежнему называются `nikki-unblock` — для совместимости.)

Настраиваешь один раз — дальше всё кнопками в веб-панели роутера, без командной строки.

## Что умеет

- **Разблокировать популярные сервисы** — по одному тумблеру (Telegram, YouTube, Netflix, ChatGPT,
  Instagram, …).
- **Добавлять свои сайты** — отправлять любой домен через VPN, оставлять напрямую или блокировать.
- **Добавлять свои VPN-серверы** — вставь ссылку-подписку или конфиг `vless://` / WireGuard; можно менять
  порядок и включить авто-выбор самого быстрого. Конфиги друзей тоже импортируются.
- **Выводить отдельные устройства мимо VPN** — исключи нужные устройства (выбрав по имени), чтобы их
  трафик шёл напрямую, а не через туннель.
- **Обходить DPI без VPN (zapret2)** — для сайтов, которые режет «изнутри» твой провайдер (YouTube и т.п.):
  включай/выключай zapret2, добавляй домены для обхода или дай ему учиться самому.
- **Одна страница управления** — включить/выключить каждый движок, обновиться одной кнопкой, бэкап настроек.

## Два движка

Nipret — «комбайн» над двумя независимыми инструментами, ставь любой из них или оба:

- **VPN (nikki / Mihomo)** — гонит выбранные сайты через прокси/VPN. Для того, что блокируют **снаружи**
  (гео-блок, блок по IP): ChatGPT, Instagram, Telegram, …
- **Обход DPI (zapret2)** — обходит DPI провайдера на уровне пакетов, без VPN. Для того, что режут
  **изнутри**: YouTube, Discord, …

Вкладки и управление движком появляются, только если он установлен.

## Перед началом

Поставь хотя бы один движок (можно оба):

- для **VPN** — **[nikki](https://github.com/nikkinikki-org/OpenWrt-nikki)**: добавь VPN и убедись, что
  подключается;
- для **обхода DPI** — **[zapret2](https://github.com/bol-van/zapret)**.

Дальше ставь Nipret — вкладки зажгутся под то, что есть.

## Установка

Выбери, что удобнее.

### Вариант A — через веб-панель роутера (без командной строки)

В LuCI зайди в **System → Software** и дальше либо:

- нажми **«Upload Package…»** и загрузи файл пакета со
  [**страницы Releases**](https://github.com/sketso/nikki-unblock/releases/latest)
  (`luci-app-nikki-unblock.apk` на OpenWrt 25.12+ или `.ipk` на старых роутерах); **либо**
- в поле **«Download and install package»** вставь эту ссылку и нажми **OK**:

  ```
  https://github.com/sketso/nikki-unblock/releases/latest/download/luci-app-nikki-unblock.apk
  ```

Роутер предупредит, что пакет из **«untrusted source»** (недоверенный источник) — это нормально (его нет
в официальном репозитории OpenWrt). Просто подтверди.

### Вариант B — одна команда (заодно включает авто-обновления в один клик)

```sh
wget -O - https://github.com/sketso/nikki-unblock/raw/main/feed.sh | sh
```

Добавляет небольшой **подписанный** фид обновлений — новые версии потом появляются прямо в
**LuCI → System → Software**, и работает `apk upgrade`.

---

В любом случае открой в веб-панели: **LuCI → Services → «nikki · Unblock»**
(или зайди на `http://IP-РОУТЕРА/nikki`).

## Обновление

- В приложении: вкладка **«Управление» → «Обновить nikki-unblock»** (работает при любом способе установки).
- Или в панели роутера: **LuCI → System → Software** — если ставил Вариантом B, он там в списке; если
  Вариантом A — просто повтори загрузку/вставку ссылки с новой версией.

## Что-то не работает?

- **Сервис включён, но через VPN ничего не идёт.** Приложение показывает предупреждение сверху, если не
  находит группу твоего VPN (ищет группу с именем `PROXY` — это стандарт). Если у твоего VPN в nikki
  группа названа иначе, открой **«Управление» → расширенные настройки** и укажи там имя группы — это
  разовая правка, и баннер сам подскажет, когда она нужна. Большинству это трогать не приходится.
- **Пресеты не грузятся / «списки недоступны».** Роутер не достаёт хост со списками — проверь
  интернет/DNS роутера или что VPN поднят.

### Настройки (`uci show nikki-unblock`)

| Опция | По умолчанию | Смысл |
|---|---|---|
| `applist_url` | `…github.io/nikki-unblock/applist` | Откуда берутся списки для кнопок-пресетов. |
| `autosync_enabled` / `autosync_interval` | `0` / `24h` | Авто-обновление включённых списков-пресетов. |
| `lang` | `ru` | Язык интерфейса по умолчанию (`ru`/`en`); переключается в браузере тумблером RU/EN. |
| `base_group` | `PROXY` | Имя VPN-группы (proxy-group) твоего профиля nikki. Меняй, только если она не `PROXY`. |
| `exit_group` | `UNBLOCK` | Имя управляемой группы (профиль + добавленные ноды), через которую идёт «→ VPN». |

Конфиги VPN-нод (ключи, токены подписок) хранятся **только на роутере**, в браузер не отдаются и никуда
не загружаются.

### Как это устанавливается (для любопытных)

На OpenWrt ≥ 25.12 установщик добавляет небольшой **подписанный** репозиторий на GitHub Pages этого
проекта (ключ кладётся в `/etc/apk/keys/`, фид — в `/etc/apk/repositories.d/`), поэтому пакет виден в
LuCI → System → Software и работает `apk upgrade luci-app-nikki-unblock`. На старых OpenWrt ставится
`.ipk` из последнего релиза. Это **не** официальный репозиторий OpenWrt — никуда ничего не подаётся, это
личный фид, как и у самого nikki.

### Свой хостинг пресетов

Пресеты берутся со статического хоста (см. [`applist/`](applist/)): `index.json` + опциональные `.lst`.
Форкни репозиторий, поправь `applist/`, включи GitHub Pages и укажи `applist_url` на свою копию. Пример
записей манифеста:

```json
{ "id": "netflix",  "name": "Netflix",  "geosite": "netflix", "geoip": "netflix" }
{ "id": "telegram", "name": "Telegram", "geosite": "telegram", "ipcidr": ["91.108.0.0/16"] }
{ "id": "torrents", "name": "Torrents" }
{ "id": "ads",      "name": "Реклама → блок", "geosite": "category-ads-all", "node": "REJECT" }
```

### Сборка из исходников

```sh
echo "src-git nikki_unblock https://github.com/sketso/nikki-unblock.git" >> feeds.conf.default
./scripts/feeds update nikki_unblock && ./scripts/feeds install -a -p nikki_unblock
make package/luci-app-nikki-unblock/compile
```

### Роутеры с малой RAM

Каждый reload VPN кратко держит две копии Mihomo (~100 МБ каждая). На слабых роутерах добавь zram:
`apk add kmod-zram zram-swap`.
