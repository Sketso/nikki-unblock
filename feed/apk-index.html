nikki-unblock — apk repository

A small signed OpenWrt apk feed for nikki-unblock. Easiest to add via the installer:

wget -O - https://github.com/sketso/nikki-unblock/raw/main/feed.sh | sh

…or manually:

wget -qO /etc/apk/keys/nikki-unblock.pem https://sketso.github.io/nikki-unblock/apk/nikki-unblock.pem
echo https://sketso.github.io/nikki-unblock/apk/packages.adb > /etc/apk/repositories.d/nikki-unblock.list
apk update && apk add luci-app-nikki-unblock
  • packages.adb — signed package index
  • nikki-unblock.pem — public signing key
  • @APKFILE@ — the package
